# 🔐 Synent PassGen — Task 4

> **Synent Technologies Python Development Internship**

A cyberpunk-themed secure password generator built with Python and Tkinter.

## ✨ Features

- 🔠 Uppercase, Lowercase, Digits, Symbols — all configurable
- 📏 Adjustable length (6–64 characters) with live slider
- 🚫 Exclude ambiguous characters (0, O, l, I)
- 💪 Real-time **strength meter** (Weak → Fair → Strong → Ultra)
- 📋 One-click copy to clipboard
- 🕓 Password history (last 4 generated)
- ⚡ Live regeneration on option change
- 🎨 Neon cyberpunk dark UI

## 🚀 How to Run

```bash
# No external dependencies — uses standard library only
python passgen.py
```

## 🛠️ Tech Stack

- Python 3.x
- Tkinter (built-in GUI)
- `random`, `string`, `re` (standard library)
